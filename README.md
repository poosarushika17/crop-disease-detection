Crop diseases continue to be a major challenge to global food security, causing 
significant yield losses and economic damage to farmers. Traditional manual disease 
diagnosis is time-intensive and prone to errors, highlighting the need for automated 
solutions. In this work, we propose an image-based crop disease detection system that 
combines Convolutional Neural Networks (CNN) using EfficientNetB0 and Vision 
Transformers (ViT) for robust and accurate classification. The EfficientNetB0 model, a 
lightweight and efficient CNN architecture, extracts fine-grained local features such as 
texture and color variations, while the ViT leverages self-attention mechanisms to 
capture long-range dependencies and global context from image patches. 
The model is trained and validated using the widely used PlantVillage dataset along 
with the PlantDoc dataset, which contains real-world plant disease images captured 
under varying environmental conditions. The combination of these datasets improves 
the model’s ability to generalize across multiple crop species and diverse field 
conditions. Once a disease is detected, the system provides fertilizer recommendations 
with both images and descriptive instructions, including dosage, application frequency, 
and safety precautions. 
To enhance accessibility for farmers, a multilingual voice assistant is integrated, 
capable of delivering the recommendations in English, Hindi, and Telugu using text-to
speech technology. By integrating state-of-the-art deep learning techniques with 
practical advisory features, this system offers a novel, scalable, and user-friendly 
approach for crop disease management. It empowers farmers with accurate diagnosis 
and actionable remedies, thereby reducing crop losses, improving productivity, and 
contributing to sustainable agriculture. 
Key words: Crop Disease Detection, Convolutional Neural Networks (CNN), 
EfficientNetB0, Vision Transformers (ViT), Image Classification, Deep Learning, 
PlantVillage Dataset, PlantDoc Dataset, Fertilizer Recommendation System, 
Multilingual Voice Assistant, Precision Agriculture, Sustainable Farming.
